
Потрібно зверстати блок контактів, використовуючи psd макет та скрін що додаються.

- mobile first
- адаптацію під desktop зробити на свій смак
- іконки блоку Contact (відсутні в psd) підібрати схожі
- карту зробити використовуючи Maps Embed API (за бажанням Maps Static API або Maps JavaScript API)
- стилі карти дефолтні, але тінь блока на декор що ліворуч повинні бути (див. psd макет)
- на mobile блоки відкриваються - закриваються по кліку на заголовок, перший відкритий за замовчуванням
